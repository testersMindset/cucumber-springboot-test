# cucumber-springboot-test

Cucumber JVM Automation Framework based on the Java Springboot

Following libraries are used in this project.

Maven Dependencies:
1) spring-boot-starter
2) spring-boot-configuration-processor
3) lombok
4) cucumber-java
5) cucumber-junit
6) cucumber-spring
